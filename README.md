# 🎮 BODMAS Premium Game

An interactive web-based game designed to teach and reinforce the **BODMAS rule (Order of Operations)** in a fun and engaging way.

## 📌 Features

🎯 Score tracking system  
🎮 Difficulty levels (Easy / Medium / Hard)  
🔊 Sound effects (Correct / Wrong)  
🎉 Confetti animation on game completion  
📱 Fully responsive design  
🎨 Modern UI with animations  

---

## 🧠 Concept Used

The game is based on the **BODMAS rule**:
- **B** – Brackets  
- **O** – Orders (Powers/Roots)  
- **D** – Division  
- **M** – Multiplication  
- **A** – Addition  
- **S** – Subtraction  

Players must select the correct operation to solve first based on priority.
